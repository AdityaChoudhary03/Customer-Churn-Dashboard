# Customer-Churn-Dashboard

## Customer Churn Analysis

This project focuses on analyzing **customer churn** and identify which customer are leaving and why.

The goal of the project is to help businesses:
- reduce churn
- retain customers
- improve customer experience
- design targeted retention strategies

---

## ⭐ Key Highlights

- ✔️ Real-world customer churn dataset
- ✔️ Data cleaning performed in Excel
- ✔️ Interactive dashboard built in Power BI
- ✔️ KPI-driven churn insights
- ✔️ Customer segmentation and risk identification
- ✔️ Strong business recommendations

---

## 🎯 Business Questions Answered

- How many customers did churn?
- Which service has the highest churn rate?
- Does low tenure increase churn?
- Do high-balance customers also churn?
- How do products and tenure affect churn?

---

## 🛠️ Tools & Technologies

- Microsoft Excel – data cleaning & preparation    
- Power BI – dashboard & visualization  
- GitHub – project documentation  

---


## 📊 Power BI Dashboard Features

- Overall churn KPI
- Churn % by Tenure band
- Churn by Risk
- Churn by tenure
- High-balance customer churn metric
- Drill-down and interactive filters

---

## 🔍 Key Insights

- Overall churn rate is approximately *26.5*
- *Month To Month customers churn significantly more*
- A considerable share of *6-12 months tenure are also churning*
- Customers are not taking all the multiple services
